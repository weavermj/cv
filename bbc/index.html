Introduction

The BBC Pagination Device can be used to cycle through breaking news feeds that are fed from a JSON source.

The component has been designed to comply with both the BBC GEL Pattern for pagination and the specific brief as given in the interview task (with a couple of assumptions).

I have implemented the functionality as a re-usable jQuery plugin. Full details of the plugin's specification are listed under the read me page.

Example

An initial example is provided below, which combines news articles from the live BBC Sport and BBC News JSON ticker feeds, with a hyper link to the full article if it is available. A title is also displayed (which is a configurable option), taken from the "prompt" attribute within the JSON data.

Further examples are available on the live demos and static demos pages.

Features

Reusable
The device can be configured at run-time to paginate items from one or more URLs of a BBC news feed in JSON format, with a news-type filter option to show only breaking news items from across all feeds
Multiple feeds per device
The device can combine multiple feeds together into a single pagination component, to show news articles from a range of different news feeds from across the BBC website (e.g. News and Sport)
Multiple devices per page
Support for multiple instances of the pagination device within the DOM is built-in, for the purpose of separately paginating multiple news feeds from different sources on a single page, if required
Auto refresh
The JSON data is automatically refreshed every 60 seconds by default, and the interval is configurable at run-time
Responsive
The UI of the pagination device reacts to changes in display screen size, to aid usability on mobile devices, by repositioning the previous and next navigational controls below the numbered page navigation
