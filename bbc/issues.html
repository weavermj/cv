Known Issues

Outstanding issues relating to functionality or browser compatibility are listed on this page.

Internet Explorer - Cross-Domain Requests

Internet Explorer 8 and up uses a different version of the XMLHttpRequest object called XDomainRequest. jQuery uses the XMLHttpRequest object when getting JSON data via ajax, which supports cross-domain requests, but in IE8+ cross-domain requests are not supported with jQuery in this manner.

The BBC Pagination Device plugin could be refactored to detect if the request was being made within Internet Explorer and provide an alternative method of obtaining the JSON source file using the XDomainRequest object. If the pagination device were to be rolled into a final released version, I would strongly recommend adding this support.

If the JSON source files were within the same domain as the website serving the jQuery plugin, then Internet Explorer should permit the request as there would be no cross-domain fetching taking place.

If you are viewing these pages in Internet Explorer 8 or above, then unfortunately the pagination devices showing live JSON data from the BBC website will not display.
